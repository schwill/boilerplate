Schwill - Boilerplate
=======

## Installation
- Install [node.js & npm](http://blog.npmjs.org/post/85484771375/how-to-install-npm)
- `$ npm i -g bower`
- `$ npm i -g gulp`

## Initial Setup

- `$ cd boilerplate`
- `$ bower i`
- `$ npm i`
- `touch .env`

## Development

- `$ npm start`

## Clean public directory

- `$ npm run clean`

## Build public directory

- `$ npm run build`
